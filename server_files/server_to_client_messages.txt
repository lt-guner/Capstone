Server-to-Client Messages:                          Meaning:

'Connected as player #'                             Successfully connected to server.
e.g. 'Connected as player 0'                        #0: client is white
                                                    #1: client is black

'Waiting on another player'                         Received by player 0 before a player 1 has connected.

'Ready'                                             Game has started. Waiting on the turn player to send a move.

'Error'                                             Failed to connect to server. Server already has two players.
